Smart Attendance System Using Face Recognition

This is a smart and efficient system to automatically mark student attendance by analyzing a classroom photo — typically captured from a CCTV camera or even a decent smartphone camera. The system identifies student faces, matches them with the dataset, and marks them as present or absent. It also generates detailed reports and can notify parents via WhatsApp if their child is absent.

Key Features

  • Face Recognition: Powered by MobileNetV2 with T2D embedding generation.
  • Face Detection: Based on OpenCV's SSD + ResNet model.
  • Attendance Report: Automatically marked as Present and Absent.
  • Parent Notifications: Sends WhatsApp updates to parents of absentees.
  • Visual Feedback: Highlights each recognized face with name and confidence score in BBox.
  • Interactive Web Interface: Upload images and view results instantly.

Image Requirements

  • Resolution: Minimum 480p if good lighting available or else 720p must.
  • Max File Size: Must be under 4 MB.
  • Student Capacity: Up to 15–20 students per image.
  • Lighting: Good lighting improves recognition accuracy.
  • Camera Angle: If placed in the corner, captures up to ~90° field of view effectively.

Dataset Info

  • Each student has ~10-15 original images.
  • With augmentation, each has 40–50 images total.
  • All images are organized in folders named after each student.

Input: Live Camera Feed

System Outputs

Face Detection & Recognition

The system detects faces in real-time and matches them against the enrolled database, displaying recognized individuals with their names and confidence scores.

Attendance Report

Generates comprehensive attendance reports with timestamps, present/absent status, and analytics dashboard for easy tracking and management.
